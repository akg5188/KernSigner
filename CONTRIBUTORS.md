# Contributors And Acknowledgements

KernSigner is a combined ESP32-P4 wallet research firmware. It is based on
Kern and also uses ideas, flows, source code, test vectors, documentation, or
integration notes from several open-source hardware-wallet and smart-card
projects.

GitHub's right-side **Contributors** panel is generated only from commit
history. It cannot be manually edited from a file. After pushing this
repository, GitHub will show commit authors that exist in this repository's
history. Imported code, local reference copies, submodules, and design
influences are acknowledged here so their provenance is still visible.

## Updating This File

To refresh the machine-derived names:

```bash
git shortlog -sne --all
git submodule foreach --recursive 'git shortlog -sne --all || true'
```

link, commit, or release reference.
