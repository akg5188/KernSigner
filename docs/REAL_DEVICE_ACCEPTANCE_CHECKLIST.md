# 整机真机验收清单

日期：2026-05-20

这份清单用于每次出固件后真机点测。没有全部完成前，只能说“继续验收”，不能说商业交付。

## 验收前先读

- [零基础第一次上手教程.zh-CN.md](零基础第一次上手教程.zh-CN.md)：适合第一次拿到设备的人按步骤走通。
- [全功能操作总手册.zh-CN.md](全功能操作总手册.zh-CN.md)：确认每个入口对应的能力和安全边界。
- [BUILD_FLASH_DEBUG_GUIDE.zh-CN.md](BUILD_FLASH_DEBUG_GUIDE.zh-CN.md)：确认构建、刷机、日志和固件 SHA256。
- [TROUBLESHOOTING_GENERAL.zh-CN.md](TROUBLESHOOTING_GENERAL.zh-CN.md)：遇到黑屏、触摸、相机、扫码、刷机问题时先查。

## 基础硬件

- 开机能进入 PIN 或首页。
- 屏幕不黑屏、不花屏。
- 触摸坐标正常。
- 返回按钮和回首页路径正常。
- 亮度设置可调，最低亮度不会误以为关机。
- 3 分钟锁屏、10 分钟关机策略按当前产品要求验证。
- 三次 PIN 错误保护正常。

## 相机和二维码

- 首次使用或拆装摄像头后，先用普通二维码在 10-20 cm 距离手动调 OV5647 镜头焦距。
- 普通文本二维码能扫码。
- URL 二维码能扫码。
- SeedQR/PSBT/签名请求从对应钱包入口进入，不从普通工具误签。
- 连续扫码两次，相机不黑屏。
- 扫码后返回，再进入扫码，相机可恢复。
- 二维码生成页删除、空格、换行、完成键正常。

## 钱包主流程

- 创建测试助记词。
- 手动导入测试助记词。
- 序号导入测试助记词。
- 十六进制、骰子、扑克牌等熵输入与 BIP39 预期一致。
- 查看指纹和地址。
- 自定义路径查看地址。
- 扩展公钥显示和二维码。
- 助记词备份、二维码备份、原始熵二维码。
- 清除会话后秘密材料不保留。
- 重启后无状态钱包不自动恢复敏感材料。

## 比特币签名

- 测试 PSBT 能解析。
- 收款地址、找零、手续费能显示。
- 非本钱包输入拒签。
- 网络 coin type 不匹配拒签。
- 签名后二维码或导出结果可被外部工具验证。
- 取消和返回不保留敏感数据。

## Web3

- OKX 连接码可扫码添加观察账户。
- Bitget 连接码可扫码添加观察账户。
- MetaMask/Rabby/TokenPocket 显示连接码，不支持的场景要有中文提示。
- 测试资金 EVM 转账签名成功。
- personalSign 样本成功或明确记录不支持。
- TypedData 必须拒签。
- 签名结果二维码能被目标钱包扫描。

## 智能卡

- 按 `SMARTCARD_REAL_DEVICE_ACCEPTANCE.md` 单独验收。
- 未外接供电时错误提示明确。
- 插拔读卡器、拔卡、断 Hub 不蓝屏。

## 留档

- 固件版本。
- 构建时间。
- SHA256。
- `boot.log`。
- 验收截图或照片。
- 失败项和复现步骤。
