# 多专家文档审阅汇总

日期：2026-05-22

这份文件记录本次文档补齐时使用的多角色审阅口径。它不是安全审计报告，也不是生产认证，只是为了说明最终用户文档按哪些模块检查过，哪些能力仍必须写成测试资金版。

## 参与审阅角色

本轮按 11 个专家角色拆分：

| 序号 | 角色 | 关注范围 |
| ---: | --- | --- |
| 1 | 安装烧录与发布包专家 | 固件下载、SHA256、full/app 刷机、回滚、首刷验收 |
| 2 | 钱包与签名专家 | 助记词、BTC 地址、公钥、PSBT、Web3 连接和签名 |
| 3 | 智能卡/Satochip/SeedKeeper 专家 | CCID、Satochip、SeedKeeper、PIN、写卡、重置、边界 |
| 4 | 硬件/屏幕/相机/SD/OTG 专家 | `wave_43` 硬件、接口、相机、SD、外壳、供电 |
| 5 | 商业钱包兼容专家 | OKX、Bitget、MetaMask、Rabby、TokenPocket、WalletConnect 参考资料 |
| 6 | 安全与生产交付专家 | PIN、安全门禁、Secure Boot 路线、测试版声明、发布证据 |
| 7 | QR/扫码/相机专家 | SeedQR、UR、BBQR、PSBT、扫码失败和相机调优 |
| 8 | 菜单覆盖/功能目录专家 | 首页菜单、子菜单、命名漂移、功能树 |
| 9 | 排错与售后专家 | 黑屏、触摸、相机、刷机、钱包、智能卡排错 |
| 10 | 文档编辑/小白可用性专家 | 术语统一、照抄步骤、结构、避免夸大能力 |
| 11 | 验收清单/最终交付专家 | 首刷验收、真机证据、生产版禁用边界 |

## 统一口径

| 主题 | 统一写法 |
| --- | --- |
| 项目状态 | 测试资金验收版，不是已审计商业生产钱包 |
| 支持硬件 | 只支持 `Waveshare ESP32-P4-WiFi6-Touch-LCD-4.3 / wave_43` |
| 开发板 PIN | 保护 Kern 设备和本机会话 |
| 智能卡 PIN | 保护 Satochip/SeedKeeper 卡 |
| 智能卡供电 | ACR39U-NF 等读卡器必须优先使用带供电 OTG 或外接供电 Hub |
| Satochip | 签名钱包卡，Web3 连接/签名按测试卡流程 |
| SeedKeeper | 秘密保存卡，写入/查看/重置都是高风险维护 |
| BTC 卡签名 | Satochip BTC PSBT/消息卡签名不作为生产开放能力 |
| TypedData/EIP-712 | 当前不能按完整生产能力宣传 |
| Secure Boot | 文档里是生产路线和门禁，不写成当前已完成证明 |

## 禁止夸大的句式

不要写：

```text
已可放心保存真实资产
商业生产级钱包
完整支持所有 EVM 链
完整支持所有 Satochip 功能
Secure Boot 已完成生产锁定
SeedKeeper 重置无风险
```

推荐写：

```text
测试资金验收版
已接入测试主线
当前验证构建可测
未完成第三方审计前不要用于真实资产
生产使用前必须通过商业发布门禁
```
