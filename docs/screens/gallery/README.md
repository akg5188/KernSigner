# 常用界面截图

这些截图来自 480x800 桌面模拟器页面，用于 GitHub 预览、文档说明和新手找入口。

注意：截图只表示界面流程可以展示，不代表所有功能已经完成真机验证、资金安全验证或生产审计。真实资产使用前必须先完成项目里的安全检查和真机验收。
