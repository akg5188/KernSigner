## 异常路径

- 不插读卡器：必须中文报错，不蓝屏。
- 读卡器无外接供电：必须中文报错，不蓝屏。
- 错 Satochip PIN：必须中文报错，不泄露 PIN。
- 连续签名两次：第二次相机必须能打开并扫码。
- 签名中返回：必须禁止离页或安全取消，不能后台任务覆盖新页面。
- 拔卡/Hub 掉电/APDU 超时：必须中文报错，后续重新插入能恢复。

## 证据保存

每次验收建议新建目录：

```bash
docs/logs/smartcard_YYYYMMDD_HHMMSS/
```

目录内至少保存：

- 串口日志。
- 测试钱包截图或录屏。
- 设备屏幕截图。
- 地址、xpub/zpub、请求类型、结果二维码扫码文本。
- 固件 SHA256 和 commit。
