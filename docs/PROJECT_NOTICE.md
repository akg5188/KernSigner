# Project Notice

Kern is an AI-assisted experimental firmware project.

It is not a finished production wallet.
It is not audited for real-funds use.
It is intended for learning, discussion, and test-only validation.

Do not store real seed phrases, private keys, or real money on it.
Do not use it for mainnet custody.
Do not treat a passing demo, screenshot, or partial smart-card flow as production readiness.

The repository contains a mix of original code, adapted code, and third-party components. See [THIRD_PARTY.md](THIRD_PARTY.md) for attribution and licenses.
